1) What is the definition of server port?

A server port is how a server machine makes its services available to the internet. It uses numbered ports, one for each service that is available on the servier. Clients can connect to a service available with a specific IP address via a specific port.

2) What is server port range?

Server ports can range from 0 to 65535.
